4. Petunjuk

  • GT dipakai sebagai patokan notifikasi jika total berat cargo melebihi GT kapal.
  • DWT tetap dapat diisi sebagai data tambahan kapal.
  • Tekan Buat Kapal Baru untuk menambah kapal custom.
  • Isi GT dan DWT kapal custom agar laporan terlihat profesional.
  • Shortcut cargo: Delete untuk hapus cargo terpilih dan Ctrl + D untuk duplicate cargo.
  • Klik area kosong lalu tarik mouse untuk memilih banyak cargo sekaligus.
  • Di HP: ketuk 2× cepat di area deck kosong lalu tahan-tarik jari untuk memilih banyak cargo sekaligus (sekali sentuh-tarik biasa dipakai untuk geser tampilan/pan).
